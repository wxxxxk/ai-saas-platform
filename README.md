# AI SaaS Platform

모듈형 AI SaaS Platform Core MVP 프로젝트입니다.

## Structure
- frontend: Next.js frontend
- backend: Spring Boot backend
- docs: architecture, product, api, database, setup docs
- prompts: Claude/Codex prompt assets
- scripts: local/dev/db scripts
- infra: deployment and infrastructure configs

## Goal
특정 AI 서비스 하나가 아닌, 다양한 AI 서비스를 Module 형태로 연결할 수 있는 Core Platform을 구축합니다.

## Core MVP
- Auth
- User/Profile
- Credit/Plan
- Job lifecycle
- Asset storage metadata
- History/Library
- Module integration structure
